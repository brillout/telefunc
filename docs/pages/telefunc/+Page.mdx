**Environment**: server.

We install Telefunc by using Telefunc's server middleware `telefunc()`:

```ts
// server.ts
// Environment: server

import { Hono } from 'hono'
import { telefunc } from 'telefunc'

const app = new Hono()

app.all('/_telefunc', async (c) => {
  const httpResponse = await telefunc({
    // Pass the web standard Request object
    request: c.req.raw,
    // Optional
    context: {
      /* Some context */
    }
  })
  return new Response(httpResponse.body, {
    status: httpResponse.statusCode,
    headers: { 'content-type': httpResponse.contentType },
  })
})
```

With Express/Fastify, pass a Node.js [`Readable`](https://nodejs.org/api/stream.html#readable-streams) stream:

```ts
const httpResponse = await telefunc({
  url: req.originalUrl,
  method: req.method,
  readable: req,
  contentType: req.headers['content-type'] || '',
})
```

The `context` parameter is optional and is only needed if we use `getContext()`, see <Link href="/getContext#provide" />.
