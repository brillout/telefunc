Telefunc integrates with any server. For production deployments with streaming, channels, and WebSocket support, use the runtime-specific `telefunc()` factory.


## Node.js

### With Hono, Fastify, or any `Request`-based server

```ts
// server.ts
// Environment: server

import { Hono } from 'hono'
import { serve } from '@hono/node-server'
import { telefunc } from 'telefunc/node'

const app = new Hono()
const tf = telefunc()

app.all('/_telefunc', async (c) => {
  const response = await tf.serve({ request: c.req.raw })
  return response ?? c.notFound()
})

const server = serve({ fetch: app.fetch, port: 3000 })

// Enable WebSocket support for channels
tf.installWebSocket(server)
```

### With Express

```ts
// server.ts
// Environment: server

import express from 'express'
import { telefunc } from 'telefunc/node'

const app = express()
const tf = telefunc()

app.all('/_telefunc', async (req, res, next) => {
  const handled = await tf.serve({ req, res })
  if (!handled) next()
})

const server = app.listen(3000)

// Enable WebSocket support for channels
tf.installWebSocket(server)
```

> The Node.js adapter auto-detects your HTTP server from the request socket. `installWebSocket()` is idempotent — calling it multiple times is safe.

### With context

```ts
app.all('/_telefunc', async (c) => {
  const response = await tf.serve({
    request: c.req.raw,
    context: {
      user: await getUser(c.req),
    },
  })
  return response ?? c.notFound()
})
```

> Access `context` inside telefunctions via <Link href="/getContext">`getContext()`</Link>.


## Bun

```ts
// server.ts
// Environment: server

import { telefunc } from 'telefunc/bun'

const tf = telefunc()

Bun.serve({
  port: 3000,
  // Pass tf.websocket to enable WebSocket channels
  websocket: tf.websocket,
  async fetch(request, server) {
    const response = await tf.serve({ request, server })
    if (response) return response

    return new Response('Not found', { status: 404 })
  },
})
```


## Deno

```ts
// server.ts
// Environment: server

import { telefunc } from 'telefunc/deno'

const tf = telefunc()

Deno.serve({ port: 3000 }, async (request, info) => {
  const response = await tf.serve({ request, info })
  if (response) return response

  return new Response('Not found', { status: 404 })
})
```


## Cloudflare Workers

```ts
// worker.ts
// Environment: server

import { telefunc } from 'telefunc/cloudflare'

const tf = telefunc()

export const TelefuncDurableObject = tf.TelefuncDurableObject

export default {
  async fetch(request: Request, env: Env, ctx: ExecutionContext) {
    const response = tf.serve({ request, env, ctx })
    if (response) return response

    return new Response('Not found', { status: 404 })
  },
}
```

Add the Durable Object and KV bindings to your `wrangler.jsonc`:

```jsonc
{
  "durable_objects": {
    "bindings": [
      { "name": "TelefuncDurableObject", "class_name": "TelefuncDurableObject" }
    ]
  },
  "kv_namespaces": [
    { "binding": "TelefuncKV", "id": "your-kv-namespace-id" }
  ],
  "migrations": [
    { "tag": "v1", "new_classes": ["TelefuncDurableObject"] }
  ]
}
```

> See <Link text="Cloudflare Workers" href="/cloudflare" /> for scaling, distributed pub/sub, delivery guarantees, and Durable Object configuration.


## Return value

`tf.serve()` returns:

| Runtime | Returns | Meaning |
|---|---|---|
| Node.js (`{ request }`) | `Response \| undefined` | `undefined` if not a Telefunc request |
| Node.js (`{ req, res }`) | `boolean` | `true` if handled, `false` if not a Telefunc request |
| Bun | `Response \| undefined` | `undefined` if not a Telefunc request |
| Deno | `Response \| undefined` | `undefined` if not a Telefunc request |
| Cloudflare | `Promise<Response> \| undefined` | `undefined` if not a Telefunc request |

> `tf.serve()` returns `undefined` (or `false`) for non-Telefunc requests, allowing you to chain it with other handlers.


## Low-level API

If you need direct control over the HTTP response, you can use the core `serve()` function:

```ts
import { serve } from 'telefunc'

app.all('/_telefunc', async (c) => {
  const httpResponse = await serve({ request: c.req.raw })
  return new Response(httpResponse.getReadableWebStream(), {
    status: httpResponse.statusCode,
    headers: httpResponse.headers,
  })
})
```
