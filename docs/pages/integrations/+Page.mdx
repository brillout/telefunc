Telefunc has a small ecosystem of first-party integrations for popular libraries and runtimes.

> **Status — first-party only.** The extension API is unstable and currently intended only for the integrations listed below. It will be opened up and documented for third-party integrations once it stabilizes.


## `@telefunc/rxjs`

Pass [RxJS](https://rxjs.dev/) `Observable` and `Subject` transparently between client and server — both directions, all operators.

→ <Link href="/integrations/rxjs" text="Full documentation" />


## `@telefunc/tanstack-query`

Live queries for [TanStack Query](https://tanstack.com/query) — server-side invalidation that automatically refetches every connected client with a matching query key.

→ <Link href="/integrations/tanstack-query" text="Full documentation" />


## `@telefunc/redis`

Redis-backed scaling for Telefunc — pub/sub fan-out across instances, plus cross-instance routing for channels.

→ <Link href="/integrations/redis" text="Full documentation" />
